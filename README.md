# Markdown WYSIWYG Editor

Editor Markdown/WYSIWYG in un singolo file: [index.html](C:\Users\giuseppe.levi\Documents\New project\index.html)

## Uso rapido

Apri `index.html` in un browser moderno.

Funzioni principali:

- editor visuale e Markdown sincronizzati
- anteprima HTML live
- apertura di file `.md`
- collegamento di una cartella progetto per immagini e risorse relative
- inserimento immagini
- pannello simboli
- interfaccia IT/EN

## Flusso consigliato per file con immagini locali

1. Clicca `Collega cartella`
2. Seleziona la cartella del progetto o del documento Markdown
3. Clicca `Apri .md`
4. Apri il file Markdown desiderato

Se il browser lo supporta, il selettore file verra aperto partendo dalla cartella gia collegata.

## Note

- Il progetto resta volutamente in un solo file HTML senza dipendenze esterne.
- Alcune API del browser, come la selezione cartelle, dipendono dal supporto del browser stesso.
- Le immagini caricate dall'editor vengono inserite nel Markdown come `data:` URL.
